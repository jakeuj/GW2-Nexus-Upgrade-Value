# 官網新增繁體中文介面展示

類型：新功能

## 為什麼做

Upgrade Value 官網目前只展示 v1.0.4 的英文主畫面與語言設定，雖然文件說明支援繁體中文，卻缺少直接的繁中介面證據。專案歷史中仍保存真實且字形正常的繁中主畫面，可用來補足語言支援展示。

## 要改什麼

- 從 Git 歷史復原真實繁中主畫面，保留為獨立網站素材。
- 建立 PNG fallback 與 640／960 像素 AVIF、WebP 衍生圖。
- 將畫面區調整為英文主圖搭配繁中與設定兩張輔助圖。
- 清楚說明英文是安全預設，繁中需使用支援 CJK 的 Nexus 字型。
- 保持繁中圖片位於首屏以下並使用 lazy loading。

## 影響範圍

- `screenshots/`：新增繁中主畫面來源。
- `docs/assets/`：新增繁中 PNG、AVIF 與 WebP。
- `docs/index.html`：新增繁中展示卡與語言支援文案。
- `docs/styles.css`：調整桌機雙列與響應式堆疊版面。
