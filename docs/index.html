First Install · Four Steps

安裝與開始掃描

  1. 1
    安裝 Nexus

    先安裝 Raidcore Nexus，並確認能在遊戲中開啟 Nexus 設定。

  2. 2
    開啟插件庫

    啟動 Guild Wars 2，在 Nexus 內開啟 Addon Library。

  3. 3
    搜尋並安裝

    找到 Upgrade Value，直接從遊戲內插件庫安裝並載入。

  4. 4
    輸入 API Key

    在插件設定貼上 Key，按「儲存並掃描」即可取得最新價格。

API Key 必要權限

在 ArenaNet API Key 管理頁建立金鑰，至少勾選以下三項。

  • account
  • inventories
  • characters
開啟 API Key 管理頁 ↗
