01

掃描整個帳號

涵蓋帳號銀行、共享物品欄、所有角色背包與目前裝備，找出物品實例中的內嵌升級。

02

三種價格並列

同時顯示立即賣、最低掛單，以及扣除 15% 交易所費用後的實際掛單價值。

03

避免拆錯裝備

標示不可分解、無交易所價格與灌注，傳奇裝備則顯示「絕對不要分解」安全警告。

04

金鑰留在本機

API Key 以 Windows DPAPI 加密保存，請求只傳送到 ArenaNet 官方 API。

四步完成

安裝與開始掃描

  1. 1
    安裝 Nexus

    先安裝 Raidcore Nexus，並確認能在遊戲中開啟 Nexus 設定。

  2. 2
    下載 DLL

    從最新 Release 下載 UpgradeValue.dll。

  3. 3
    放進 addons

    將 DLL 放到 Guild Wars 2 的 addons 根目錄，不是 addons\Nexus。

  4. 4
    輸入 API Key

    在插件設定貼上 Key，按「儲存並掃描」即可取得最新價格。

API Key 必要權限

在 ArenaNet API Key 管理頁建立金鑰，至少勾選以下三項。

  • account
  • inventories
  • characters
開啟 API Key 管理頁 ↗
